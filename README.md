## About

BloomPath is a personal growth dashboard designed for students — focused on mindful
progress tracking, habit building, and self-reflection. Built with smooth Framer Motion
animations and a clean component-based React architecture.

Submitted as a **PDF entry for Frontend Battle 2026** — a frontend design competition
focused on UI craftsmanship and interaction design.

---

## Features

- Mindful growth tracking interface for students
- Smooth page and component animations with Framer Motion
- Clean, responsive layout built with Tailwind CSS
- Component-based architecture with React + Vite
- Deployed live on Vercel

---

## Getting started

```bash
# Clone the repo
git clone https://github.com/SAMYUKTHASRR/bloompath.git

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
